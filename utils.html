pyharnessworkshop.utils.k8s.add_k8s_service_to_hosts(service_name, namespace, hostname)[source]

Adds a Kubernetes service IP to the /etc/hosts file.

Parameters:

  • service_name – The name of the Kubernetes service.

  • namespace – The namespace of the Kubernetes service.

  • hostname – The hostname to map to the service IP.

pyharnessworkshop.utils.k8s.apply_k8s_manifests(manifests, namespace='default')[source]

Apply Kubernetes manifests.

Parameters:

  • manifests – The path to the manifests file(s).

  • namespace – The namespace for the Kubernetes secret. Default is ‘default’.

pyharnessworkshop.utils.k8s.create_k8s_secret(secret_name, secret_data, namespace='default')[source]

Create Kubernetes secret.

Parameters:

  • secret_name – The name of the Kubernetes secret to create.

  • namespace – The namespace for the Kubernetes secret. Default is ‘default’.

Raises:

SystemExit – If the secret could not be created.

pyharnessworkshop.utils.k8s.get_k8s_loadbalancer_ip(service_name, namespace='default', max_attempts=15)[source]

Retrieves the external IP of a Kubernetes LoadBalancer service.

Parameters:

  • service_name – The name of the Kubernetes service.

  • namespace – The namespace of the Kubernetes service. Default is ‘default’.

  • max_attempts – The maximum number of attempts to get the IP. Default is 15.

Returns:

The external IP address of the LoadBalancer service.

Raises:

SystemExit – If the IP address could not be retrieved within the maximum attempts.

pyharnessworkshop.utils.k8s.render_manifest_from_template(template_file, output_path, apps_string)[source]

Renders a Kubernetes manifest from a template by replacing placeholders with actual values.

Parameters:

  • template_file – The path to the template file.

  • output_path – The path where the rendered manifest will be saved.

  • apps_string – A comma-separated string of app details in the format ‘app_name:app_port:ip_address’.

pyharnessworkshop.utils.k8s.wait_for_kubernetes_api(k8s_api)[source]

Enables bash completion for kubectl. Waits for the Kubernetes API server to become available.

Parameters:

k8s_api – The URL of the Kubernetes API server. (e.g., ‘http://localhost:8001/api’)

Returns:
